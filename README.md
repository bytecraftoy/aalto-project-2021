# Project

## How to run

### Development mode

Front-end:

```bash
# Go inside the directory
cd src/client

# Install dependencies (No need after first time)
npm install

# Start development server
npm run start
```

Documentation:

```bash
# Go inside the directory
cd src/docs

# Install dependencies (No need after first time)
npm install

# Start development server
npm start
```

Back-end:

```bash
# Go inside the directory
cd src/server

# Install dependencies (No need after first time)
npm install

# Start development server
npm run dev
```

Postgres:

```bash
# Go inside the directory
docker-compose up -d
```

Cypress:

```bash

# Start Cypress
npx cypress open
```
